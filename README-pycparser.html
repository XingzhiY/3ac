Pycparser Setup Guide

Install pycparser

Install the latest version of pycparser. You can do this by:

pip3 install pycparser --user

Or:

python3 -m pip install pycparser --user

The source code for pycparser can be found on GitHub

Using pycparser

For this assignment, you will need to parse a C file into an AST, change it into 3 address code and write back the changed AST to the output file.

I strongly recommend reading the examples at https://github.com/eliben/pycparser/blob/master/examples/ to understand how to use pycparser.

Notably:

  1. c-to-c.py demonstrates parsing and writing back a C file.
  2. rewrite_ast.py demonstrates changing values in the AST.
  3. func_calls.py shows an example of the visitor pattern for traversing the AST.

You can also use examples/dump_ast.py to show the AST structure of a C file – this is very useful in understanding the conversion of C code to ASTs. The file pycparser/_c_ast.cfg contains the full specification for the structure of the AST nodes.

If you use the examples that contain C preprocessor directives (e.g. #include), you must run the preprocessor before you run pycparser. See the file using_cpp_libc.py in examples for code that does this.
